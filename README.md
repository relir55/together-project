# 🍵 Genshin Mugs Shop Bot

> Telegram-бот для продажи кружек с персонажами Genshin Impact — каталог, поиск и платёжная система прямо в мессенджере.

---

## 📸 Возможности

- 📋 **Каталог** — листаемый список из 100+ товаров с пагинацией
- 🔍 **Поиск** — мгновенный поиск по имени персонажа
- 💰 **Баланс** — просмотр и пополнение личного баланса
- 🧾 **История** — лог всех платежей пользователя
- 🔗 **Wildberries** — прямые ссылки на покупку в один клик

---

## 🗂 Структура проекта

```
genshin-mugs-bot/
│
```

## ⚙️ Технологии

- **Python 3.10+**
- **pyTelegramBotAPI** (`telebot`) — работа с Telegram Bot API
- **SQLite3** — хранение товаров, пользователей и платежей
- **Inline-клавиатуры** — навигация по каталогу без команд

## 🚀 Запуск

**1. Клонируй репозиторий**

```bash
git clone https://github.com/username/genshin-mugs-bot.git
cd genshin-mugs-bot
```

**2. Установи зависимости**

```bash
pip install pyTelegramBotAPI
```

**3. Настрой конфиг**

Открой `config.py` и вставь токен своего бота:

```python
TOKEN = "твой_токен_от_BotFather"
DATABASE = "shop.db"  # имя файла БД, можно оставить как есть
```

**4. Запусти бота**

```bash
python main.py
```

При первом запуске автоматически создастся файл `shop.db` и заполнится весь каталог товаров.

---

## 🤖 Команды бота

- `/start` — начать работу, регистрация пользователя
- `/catalog` — открыть каталог кружек
- `/search` — найти кружку по имени персонажа
- `/balance` — посмотреть свой баланс
- `/topup` — пополнить баланс
- `/history` — история платежей
- `/help` — список всех команд

---

## 🗃 Схема базы данных

**Каталог товаров**
- `number` — ID товара, автоинкремент
- `name` — название товара (уникальное)
- `price` — цена в рублях
- `link` — ссылка на Wildberries

**Покупатели**
- `user_id` — Telegram ID пользователя
- `username` — никнейм в Telegram
- `balance` — баланс в рублях

**Платежи**
- `id` — ID платежа, автоинкремент
- `user_id` — кому принадлежит платёж
- `amount` — сумма
- `status` — `pending` / `confirmed` / `failed`
- `created_at` — дата и время создания
